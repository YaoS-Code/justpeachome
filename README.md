# JUST PEAC HOMES

This is the monorepo for the JUST PEAC HOMES website.

## 📁 Project Structure
- `frontend/`: Next.js application (Edge Runtime)
- `studio-borui/`: Sanity Studio (Content Management)
- `docs/`: Comprehensive project documentation

## 📚 Documentation
For detailed guides on development, design, and content strategy, please refer to the `docs/` directory:

- [**STYLING_GUIDE.md**](docs/STYLING_GUIDE.md): Design tokens, colors, and typography.
- [**SEO_STRATEGY.md**](docs/SEO_STRATEGY.md): SEO and AI Search optimization strategy.
- [**SANITY_GUIDELINES.md**](docs/SANITY_GUIDELINES.md): CMS schema and usage guidelines.
- [**DEVELOPMENT_STANDARDS.md**](docs/DEVELOPMENT_STANDARDS.md): Tech stack, coding rules, and deployment.
- [**TROUBLESHOOTING.md**](docs/TROUBLESHOOTING.md): Historical error log and bug fixes.

## 🚀 Quick Start
1. **Start Sanity**: `cd studio-borui && npm run dev`
2. **Start Frontend**: `cd frontend && npm run dev`

## 📧 Infrastructure
- **Hosting**: Cloudflare Pages
- **CMS**: Sanity.io
- **Email**: Resend API
- **DNS**: Cloudflare
